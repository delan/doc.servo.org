Struct ByteReader 

Source 
pub struct ByteReader<S> {
    stream: S,
    bytes: Option<Bytes>,
}
Expand description

Treat a websocket stream as an AsyncRead implementation.

This also works with any other Stream of Message, such as a SplitStream.

Each read will only return data from one message. If you want to combine data from multiple messages into one read, consider wrapping this in a BufReader.
