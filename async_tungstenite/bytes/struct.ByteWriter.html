Struct ByteWriter 

Source 
pub struct ByteWriter<S> {
    sender: S,
    state: State,
}
Expand description

Treat a websocket sender as an AsyncWrite implementation.

Every write sends a binary message. If you want to group writes together, consider wrapping this with a BufWriter.
