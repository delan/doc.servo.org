Struct ByteWriter 

Source 
pub struct ByteWriter<W: Write, E: Endianness> {
    phantom: PhantomData<E>,
    writer: W,
}
Expand description

For writing aligned bytes to a stream of bytes in a given endianness.

This only writes aligned values and maintains no internal state.
