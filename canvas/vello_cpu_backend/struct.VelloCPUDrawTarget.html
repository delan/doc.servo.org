Fields§

§ctx: RenderContext

Because this is stateful context caller cannot assume anything about transform, paint, stroke, so it should provide it’s own used by each command but it can assume paint_transform to be identity and fill rule to be peniko::Fill::NonZero

This is because paint_transform is rarely set, so it’s cheaper to always reset it after use.
