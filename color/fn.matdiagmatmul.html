pub(crate) const fn matdiagmatmul(
    ma: &[[f32; 3]; 3],
    mb: [f32; 3],
) -> [[f32; 3]; 3]
Expand description

Multiplication ma * mb of a 3x3-matrix ma by a 3x3-diagonal matrix mb.

Diagonal matrix mb is given by

[ mb[0] 0     0     ]
[ 0     mb[1] 0     ]
[ 0     0     mb[2] ]
