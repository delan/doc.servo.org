pub fn parse_color(s: &str) -> Result<DynamicColor, ParseError>

Parse a color string in CSS syntax into a color.

This parses the entire string; trailing characters cause an ExpectedEndOfString parse error. Leading and trailing whitespace are ignored. See also parse_color_prefix.
