pub(crate) struct Conn<I, B, T> {
    io: Buffered<I, EncodedBuf<B>>,
    state: State,
    _marker: PhantomData<fn(T)>,
}
Expand description

This handles a connection, which will have been established over an Read + Write (like a socket), and will likely include multiple Transactions over HTTP.

The connection will determine when a message begins and ends as well as determine if this connection can be kept alive after the message, or if it is complete.
