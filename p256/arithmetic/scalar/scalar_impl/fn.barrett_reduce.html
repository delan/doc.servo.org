Barrett Reduction

The general algorithm is:

p = n = order of group
b = 2^64 = 64bit machine word
k = 4
a \in [0, 2^512]
mu := floor(b^{2k} / p)
q1 := floor(a / b^{k - 1})
q2 := q1 * mu
q3 := <- floor(a / b^{k - 1})
r1 := a mod b^{k + 1}
r2 := q3 * m mod b^{k + 1}
r := r1 - r2

if r < 0: r := r + b^{k + 1}
while r >= p: do r := r - p (at most twice)
