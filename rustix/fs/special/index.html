The CWD and ABS constants, representing the current working directory and absolute-only paths, respectively.

§Safety

This file uses AT_FDCWD, which is a raw file descriptor, but which is always valid, and -EBADF, which is an undocumented by commonly used convention of passing a value which will always fail if the accompanying path isn’t absolute.
