Module ioctl

The Unix ioctl function is effectively lots of different functions hidden behind a single dynamic dispatch interface. In order to provide a type-safe API, rustix makes them all separate functions so that they can have dedicated static type signatures.

Some ioctls, such as those related to filesystems, terminals, and processes, live in other top-level API modules.
