fn unpad_tls13_payload(p: &mut BorrowedPayload<'_>) -> ContentType
Expand description

Decode a TLS1.3 TLSInnerPlaintext encoding.

p is a message payload, immediately post-decryption. This function removes zero padding bytes, until a non-zero byte is encountered which is the content type, which is returned. See RFC8446 s5.2.

ContentType(0) is returned if the message payload is empty or all zeroes.
