drawImage coordinates explained

 Source Image      Destination Canvas
+-------------+     +-------------+
|             |     |             |
|(sx,sy)      |     |(dx,dy)      |
|   +----+    |     |   +----+    |
|   |    |    |     |   |    |    |
|   |    |sh  |---->|   |    |dh  |
|   |    |    |     |   |    |    |
|   +----+    |     |   +----+    |
|     sw      |     |     dw      |
|             |     |             |
+-------------+     +-------------+

The rectangle (sx, sy, sw, sh) from the source image is copied on the rectangle (dx, dy, dh, dw) of the destination canvas
