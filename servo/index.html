Servo, the mighty web browser engine from the future.

This is a very simple library that wires all of Servo‚Äôs components together as type Servo, along with a Webview implementation, WebView to create a working web browser.

The Servo type is responsible for configuring a Constellation, which does the heavy lifting of coordinating all of Servo‚Äôs internal subsystems, including the ScriptThread and the LayoutThread, as well maintains the navigation context.
