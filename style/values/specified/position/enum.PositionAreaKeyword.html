Expand description

Possible values for the position-area property’s keywords. Represented by [0z xxx yyy], where z means “self wm resolution”, xxxx is the axis (as in PositionAreaAxis) and yyy is the PositionAreaTrack https://drafts.csswg.org/css-anchor-position-1/#propdef-position-area
