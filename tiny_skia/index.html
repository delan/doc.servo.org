tiny-skia is a tiny Skia subset ported to Rust.

tiny-skia API is a bit unconventional. It doesnâ€™t look like cairo, QPainter (Qt), HTML Canvas or even Skia itself. Instead, tiny-skia provides a set of low-level drawing APIs and a user should manage the world transform, clipping mask and style manually.

See the examples/ directory for usage examples.
